Usage & API Billing

This document applies exclusively to Clients where AI functionality has been integrated into their website. It describes how API usage is measured, billed and monitored.

  1. Included usage

    Where the Client has chosen an AI integration, a base volume of API usage is included in the monthly service fee (the "Included Usage"). This volume is set out in the project quote or service agreement. Standard website services without AI integration are not covered by this document.

  2. Overage and peak usage

    API usage that exceeds the Included Usage within a billing cycle is passed on to the Client as "Overage" at the prevailing cost price of the relevant AI supplier, plus an administration fee.

    • Threshold: once API usage exceeds the Included Usage by more than 10%, smartflux sites reserves the right to invoice these additional costs directly.
    • Liability: the Client is responsible for costs arising from high end-user traffic or unexpected usage spikes, even when caused by external factors.

  3. Structural adjustments

    If the Client's API usage exceeds the Included Usage for two (2) consecutive months, smartflux sites will, in consultation with the Client, adjust the package to a higher usage threshold that better matches actual consumption. The Client will be informed in a timely manner.

  4. Monitoring and safety limits

    smartflux sites actively monitors the API usage of integrated AI features. In the event of an unusual spike indicating a technical error or abuse, smartflux sites reserves the right to temporarily pause the AI functionality to prevent further unwanted costs for both parties. The Client will be notified immediately.

  5. Payment terms

    Fixed monthly fees are invoiced in advance. Overage charges are invoiced in arrears at the end of the month, or immediately upon reaching a critical threshold. Failure to pay outstanding amounts may result in temporary suspension of the AI functionality on the Client's website.
