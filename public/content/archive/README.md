# Dutch Content Archive

## Development Strategy

During the development phase, **all Dutch (`-nl.html`) content files are archived here**.

### Why?
- Avoids duplicate work on every change
- Prevents inconsistencies between English and Dutch versions
- Makes it clear we're developing English-first

### Workflow
1. **Development Phase** (NOW): Work only on English versions
   - `home.html`
   - `about.html`
   - `case-studies.html`
   - `contact.html`

2. **Before Launch**: Translate finalized English versions to Dutch
   - Copy each English file
   - Translate all text content to Dutch
   - Replace archived versions

### Archived Files
- `home-nl.html` - Dutch homepage
- `about-nl.html` - Dutch about page
- `case-studies-nl.html` - Dutch case studies page
- `contact-nl.html` - Dutch contact page

---

**For Claude Code sessions:** Do NOT update Dutch versions during development. Work only on English files until ready for launch.
